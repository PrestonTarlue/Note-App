NoteApp-Cpp

A command-line note-taking application built to practice C++ fundamentals and Linux development workflows.
🚀 Features

    Persistent Storage: Saves notes to a local text file (note.txt).

    Menu-Driven Interface: Uses a while loop and switch statements for a clean user experience.

    File Handling: Implements fstream to manage reading and writing data directly to the disk.

🛠️ How to Build and Run

This project was developed and tested on Kali Linux using the g++ compiler.

    Clone the repository:
    Bash

git clone <your-repo-url>
cd NoteApp-Cpp

Compile the code:
Bash

g++ main.cpp -o noteapp

Run the application:
Bash

    ./noteapp

🧠 What I Learned

    Linux Terminal: Mastering directory navigation and compiler execution (g++).

    C++ I/O: Using std::cin and std::getline to handle user input, including clearing the input buffer with cin.ignore().

    File Streams: Working with std::ofstream for writing and std::ifstream for reading files.

    Control Flow: Implementing robust menu systems with while loops and switch statements.
